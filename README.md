# Sentiment Analysis on Amazon Reviews

## Project Overview
This project performs sentiment analysis on Amazon product reviews using Machine Learning and Natural Language Processing (NLP). The model classifies reviews as positive or negative based on review text.

## Features
- Text preprocessing
- Stopword removal
- Data visualization
- Feature extraction using CountVectorizer
- Machine learning classification
- Sentiment prediction
- Accuracy evaluation

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NLTK
- Scikit-learn
- XGBoost
- WordCloud

## Installation

### Clone Repository

```bash
git clone https://github.com/Darshanven/sentiment-analysis-amazon-reviews.git
```

### Move to Project Folder

```bash
cd sentiment-analysis-amazon-reviews
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Jupyter Notebook

```bash
python -m notebook
```

## Dataset
The dataset contains Amazon product reviews used for sentiment classification.
